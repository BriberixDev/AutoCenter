# Security Policy

Thanks for taking the time to report security issues.

## Supported Versions
This is a portfolio project under active development. Only the latest version on the default branch is supported.

## Reporting a Vulnerability
Please report security vulnerabilities privately.

Preferred: GitHub Security Advisories (Security tab → “Report a vulnerability”).

If that’s not available, contact me via email:
- security contact: <briberix.dev@gmail.com>

Please include:
- What you found and why it matters
- Steps to reproduce (minimal PoC if possible)
- Affected page/endpoint/component
- Any relevant logs or screenshots (do not include secrets)

Please do not open public issues for security reports.
